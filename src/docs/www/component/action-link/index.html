Action links are used to signpost the start of a user journey.

When to use this component

Use an action link to signpost the start of a clearly defined user journey.

When not to use this component

  • Do not use an action link inside a paragraph of text.
  • Do not use an action link to emphasise the importance of a link.
  • Do not use an action link in place of a button.
