Inset text helps users identify and understand the most important content on the page.

When to use this component

Use inset text to make content stand out on the page and allow users to identify key information, even if they do not read the whole page.

When not to use this component

  • Do not use inset text to signal the start of a user journey. Use an Action Link instead.
