Use these styles to make your application fit in with the NIHR ecosystem.

The Design System for the NIHR styles can be implemented in any web application, regardless of technology stack. They serve as blueprints for your own implementations, as well as the basis for component designs.

After installing and importing the Design System, these styles will be available and applied to your application out of the box. These implementations are built on top of Bootstrap and provide Nunjucks macros.

If your web application uses different technologies, you can implement these styles as well as the components yourself on top of a framework of your choice.
