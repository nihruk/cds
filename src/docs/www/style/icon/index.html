The NETSCC Design System uses the Font Awesome icon library. Its free icons are included with every installation of the Design System.

How to use icons

The easiest way to add icons is by applying Font Awesome's CSS classes.

Always add a label to each icon, describing its purpose. For icons that are easily recognized and understood by users, this label may be visually hidden.

When to use icons

Use icons for communication, such as to indicate a specific action.

You must ensure that the icons you use have a positive effect on your application's User Experience, and that its users recognize their meaning.

When not to use icons

Do not use icons for decoration.
