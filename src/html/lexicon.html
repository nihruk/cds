Design elements

Design elements are the first step in creating a comprehensive design system by defining base level elements. These elements combine to determine the components of your product.

Elements we have included:

  • Color
  • Typography
  • Sizing
  • Spacing
  • Grids

Design Components

Design components are functional pieces of a user interface and are informed by the design elements and come together in various ways to create one or more components.

Components we have included:

  • Forms
  • Buttons
  • Tabs
  • Icons
  • Alerts

Design views (screens & pages)

After you define your design elements and components, you can combine them to create views.

Examples:

  • Primary navigation bar Search bar Footer Mega menu

Include design patterns

Design patterns consist of common user flows and interactions to achieve a specific goal to achieve a task.

Examples include:

  • Filtering, e.g. showing how results are displayed
  • Engaging with social media
  • Progressive disclosure, e.g. showing the number of steps in a form and progress in completing it.
  • Continuous scrolling, e.g. a carousel to scroll through a set of products on an e-commerce site.
  • On-boarding walk-throughs, e.g. walk someone through a set of instructions
