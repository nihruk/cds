Design System for the NIHR

Design and build web applications for the NIHR that are accessible, user-friendly, and provide consistent user-journeys using the Design System's off-the-shelf components. =======

NIHR - Design System DS

The Design System comes with a front-end library that contains all the code needed to quickly and reliably use any of the components on a website of your own.

The library is based on the Bootstrap front-end framework. If this conflicts with your project requirements, you may optionally implement your own front-end code based on the component designs and documented behaviors.
