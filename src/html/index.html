Design System for the NIHR

Design and build web applications for the NIHR that are accessible, user-friendly, and provide consistent user-journeys using the Design System's off-the-shelf components.

Components in the Design System are

  • consistent with NIHR branding
  • WCAG 2.1 AA compliant
  • reviewed for optimal UX
  • often available as Bootstrap implementations through the npm package
